# 📑 SIGA V2

## 🎨 Escolhas de Design

O projeto visou modernizar a interface acadêmica do sistema SIGA, focando em uma estética funcional e intuitiva.

* **Glassmorphism:** Aplicação de superfícies translúcidas seguindo uma estética mais moderna e minimalista, suave.
* **Continuidade Visual:** A transição da Splash Screen para o Login feita de forma suave, evitando quebras na percepção do usuário.
* **Tipografia e Espaçamento:** Foco em legibilidade com pesos de fonte variados e ícones minimalistas para uma navegação intuitiva.

---

## ✨ Personalizações Realizadas

1.  **Relógio:** Exibição da hora atualizada em tempo real (formato `HH:mm`).
2.  **Tooltips:** Balões de informação personalizados via CSS, que explicam métricas acadêmicas ao passar o mouse.
